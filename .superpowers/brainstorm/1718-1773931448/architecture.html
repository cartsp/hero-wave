Data Flow

How the component works at runtime
1. Blazor renders <WavyBackground> → outputs a <div> with a <canvas> + overlay
↓
2. OnAfterRenderAsync → calls JS module via IJSRuntime
↓
3. JS receives canvas ref + config (colors, speed, blur, wave count)
↓
4. JS starts requestAnimationFrame loop with simplex noise
↓
5. Blazor content (ChildContent / Title / Subtitle) renders in overlay div on top
↓
6. On dispose → Blazor calls JS cleanup (stops animation, releases canvas)
